# TwitLife: Phase 24.5 - Avatar Evolution

TwitLife is a deep, BitLife-style social media simulation built with Next.js and Python (FastAPI). It simulates the cutthroat strategy and tribal dynamics of a modern social network.

## Project Structure

- **`/backend`**: FastAPI server handling the simulation engine, fandom math, and NPC logic.
- **`/twitlife`**: Next.js frontend providing the authentic "X-style" UI.

## Key Features (Phase 24.5)

- **Vector Space Niches**: Competitive ranking and engagement logic based on 500-trait personality vectors.
- **Sovereignty & Niche Kings**: Specialized buffs and penalties for domain-specific dominance.
- **Dynamic Avatar System**: Illustrated human portraits for 1,200+ NPCs via DiceBear.
- **Aura Rings & Verified Hexagons**: Visual indicators for social status and verification.
- **Shadow Market**: Monetize Aura and buy algorithmic influence.

## Getting Started

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn main:app --reload`

### Frontend
1. `cd twitlife`
2. `npm install`
3. `npm run dev`
